SVG – Scalable Vector Graphics (Масштабируемая Векторная Графика) – основанный на векторе формат для веб-разработчиков. Данный тип формата стал столь популярен благодаря тому, что его изображения возможно отображать на больших разрешениях без потери качества, так как SVG – это векторный формат.

SVG имеет небольшой вес по сравнению с другими форматами. У этого формата просто невероятные возможности. Флаги, символы, элементы интерфейса. И это только небольшой список того, как его можно использовать.

Рядом с этим текстом картинки современного веб-формата изображений .svg, они меньше тормозят загрузку страницы в браузере.

Толчок развития данной языковой разметке впервые был дан Консорциумом Всемирной паутины (World Wide Web Consortium), который был более известен в 1999 году как W3C. W3C дала понятие SVG – языковая разметка для создания двухмерных графических интерфейсов и изображений.

Самый большой плюс его использования, это то что он является векторным форматом, то есть его возможно использовать на любых дисплеях – каким бы ни было их разрешение, он везде будет отображаться одинаково.

Секция вопрос ответ

Нажав на вопрос вы получите полноценный ответ на интересующую вас информацию

  • Как загрузить файлы сайта на хостинг?

    Для загрузки не большого количества файлов можно обойтись администраторским разделом хостинга. Заходим в клиентскую часть Вашего аккаунта, находим "Файловый менеджер", в котором должна лежать папка с названием Вашего сайта

  • Как загрузить файлы по FTP?

    При загрузке на хостинг большого количества папок и файлов с емким содержимым (видео, музыка), используются специальные программы FTP клиента. Самый простой - Total Commander, хороший бесплатный - FileZilla Client, хороший платный - CuteFTP

  • Как настроить соединение по FTP для клиента CuteFTP?

    Адрес хоста (FTP Host Address): личный домен, например: sitey.ru
    Имя узла (Label for site): Любое имя для себя, потом будете на него нажимать для подключения
    Имя пользователя (FTP site User Name): логин на хосте (или смотрите тех. данные хоста)
    Пароль (FTP site Password): ваш пароль (пароль можно настроить в Вашей админ. части хоста)

  • Что такое индексный файл index.html?

    В любой папке необходимо наличие файла index.html (index.php, index.cgi...). Обычно этот файл является главной страницей сайта или раздела. Каждый современный браузер настроен на открытие этого файла по умолчанию. Если в папке лежит файл index.html, то браузер откроет его без соответствующего набора.

  • Для чего нужен файл style.css?

    Сайт построенный с применением технологии CSS (каскадные таблицы стилей) имеет много преимуществ и обеспечивает быстроту открытия, переходов по страницам сайта. Это достигается за счет того, что любой браузер при открытии сайта в первую очередь ищет на нем и считывает информацию из файла style.css. Название может быть любым, но расширение .css обязательно.

  • Дополнительно о файле style.css

    Файл style.css дает понять браузеру из чего состоит сайт, его размерность, где лежат картинки, какой шрифт, цвет, фон... разных страниц и разделов. Браузер записывает эту информацию к себе в кеш (временная память) и при продолжении работы с этим сайтом он повторно не обращается к серверу (провайдеру), на котором лежит Ваш сайт (папки, файлы)..
